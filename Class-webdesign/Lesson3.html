Introduction to CSS

CSS stands for Cascading Style Sheets.

Ways of including CSS codes in HTML document

There are three ways of including CSS codes in HTML :

  1. Inline CSS
  2. Internal CSS
  3. External CSS

Four types of selectors

There are 4 main ways of selecting elements with CSS:

  • Element Selector
  • Class Selector
  • ID Selector
  • Pseudo Selector
